# test
Разработка проходила на фреймворке Laravel.
После скачивания файлов необходимо установить Composer в папку с проектом: composer install.
Выполнитть миграцию в БД : php artisan migrate.
Для запуска приложения понадобится доступ к серверу phpMyAdmin.
Запуск приложения происходит через браузер. http://localhost/
